Gemma Guide is a blind-first multimodal navigation assistant, built for a Gemma hackathon around one idea: an agent is only as capable as the tools you hand it. Gemma can reason about space but can't estimate depth, so I wired in Google's TIPS vision model as a tool.

The agent “points” at an object with a bounding box, then pulls that object's depth straight from TIPS. Because TIPS also ships a segmentation head, I could pinpoint the exact thing Gemma was pointing at, not just roughly where it was. The result is a guidance agent that understands where things are in a scene, not only what they are.

// what I learned

  • An agent's ceiling is its toolset: handing Gemma a depth + segmentation tool unlocked spatial reasoning it fundamentally couldn't do alone.
  • Bounding-box “pointing” plus a segmentation head is a clean way to ground a language model to a precise object rather than a vague region.
  • Designing blind-first forces real clarity: the system has to know where things are, not just be able to name them.
