# local, privacy-preserving RAG

  • Retrieval-Augmented Generation (RAG)
  • LangGraph
  • Ollama
  • Vector Search
  • Python
> view on github

My oldest project, and where I first learned to wire multiple models together with LangGraph. It's a RAG system over my own WhatsApp and Instagram chat history: upload the export, it processes everything, and you can ask questions about your messages.

The constraint that makes it interesting is that it runs entirely locally on a Mac through Ollama, so nothing private ever leaves the machine. It lets me trace and search old conversations while keeping the data fully private by design.

// what I learned

  • My first time wiring multiple models together with LangGraph: a graph is a clean way to reason about multi-step LLM flows.
  • Running entirely locally through Ollama means private data never leaves the machine: privacy by design, not by policy.
  • RAG over personal chat history is a surprisingly good way to search and trace old conversations.
