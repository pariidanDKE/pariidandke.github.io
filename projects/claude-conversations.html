Claude Conversations

# dev tooling + cost-aware product thinking

work in progress
  • Prompt Engineering
  • LLM Summarization
  • Developer Tools

A Claude Code plugin that scratches my own itch: picking up relevant conversations from previous days without scrolling forever. It uses smart compaction to generate a title and description for each past conversation, and I built it around keeping that summarization cheap and fast, so it stays a pleasant tool rather than a costly one.

A small project, but a real exercise in balancing product quality against time and cost. (Still a work in progress; repo and screenshots coming soon.)

// what I learned

  • Keeping summarization cheap and fast is what makes a tool pleasant to use rather than costly to run.
  • Smart compaction, a title and description per past conversation, solves the “where was that chat” problem without endless scrolling.
  • A small project is still a real exercise in balancing product quality against time and cost.
